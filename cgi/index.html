CGI(common gateway interface)

CGI是服务器的标准,也就是任意具有输入输出的编程语言都可以轻易编写CGI程序

python http server

通过python的http模块作为server

以下细节要注意
CGI脚本一定要放在cgi-bin目录下
CGI脚本一定要有执行权限
